================================================================================
                           INDICATORS DEFINITION
================================================================================

┌───────────┬─────────────────────────────────────────────────────────────────┐
│ ID        │ PAI - Procedural Absence Index                                │
│ Question  │ Who has not yet been invited to this table?                   │
│ Threshold │ 0.7                                                           │
│ Formula   │ 1 - (M/T) where M = mentioned groups, T = total groups        │
└───────────┴─────────────────────────────────────────────────────────────────┘

┌───────────┬─────────────────────────────────────────────────────────────────┐
│ ID        │ CGI - Credibility Gap Index                                   │
│ Question  │ How many promises have we made, and how many have we kept?    │
│ Threshold │ 0.7                                                           │
│ Formula   │ Semi-mathematical with keyword weights                        │
└───────────┴─────────────────────────────────────────────────────────────────┘

┌───────────┬─────────────────────────────────────────────────────────────────┐
│ ID        │ ERI - Epistemic Rigidity Index                                │
│ Question  │ Is our certainty our prison?                                  │
│ Threshold │ 0.65                                                          │
│ Formula   │ certainty / (certainty + doubt)                               │
└───────────┴─────────────────────────────────────────────────────────────────┘

┌───────────┬─────────────────────────────────────────────────────────────────┐
│ ID        │ FQI - Forbidden Question Index                                │
│ Question  │ What question are we afraid to ask?                           │
│ Threshold │ 0.7                                                           │
│ Formula   │ Estimative (expert judgment)                                  │
└───────────┴─────────────────────────────────────────────────────────────────┘

┌───────────┬─────────────────────────────────────────────────────────────────┐
│ ID        │ AGI - Actor Gap Index                                         │
│ Question  │ Who is excluded from decision-making?                         │
│ Threshold │ 0.7                                                           │
│ Formula   │ Semi-mathematical with keyword weights                        │
└───────────┴─────────────────────────────────────────────────────────────────┘

┌───────────┬─────────────────────────────────────────────────────────────────┐
│ ID        │ DIC - Diversity Index                                         │
│ Question  │ How many perspectives are represented?                        │
│ Threshold │ 0.7                                                           │
│ Formula   │ Semi-mathematical with keyword weights                        │
└───────────┴─────────────────────────────────────────────────────────────────┘

┌───────────┬─────────────────────────────────────────────────────────────────┐
│ ID        │ MCI - Meta-Cognitive Index                                    │
│ Question  │ Does the text acknowledge its limits?                         │
│ Threshold │ 0.6                                                           │
│ Formula   │ Semi-mathematical with keyword weights                        │
└───────────┴─────────────────────────────────────────────────────────────────┘

┌───────────┬─────────────────────────────────────────────────────────────────┐
│ ID        │ LRI - Legislative Rigidity Index                              │
│ Question  │ How many times has this law been amended?                     │
│ Threshold │ 0.6                                                           │
│ Formula   │ 1 / (amendments + 1)                                          │
└───────────┴─────────────────────────────────────────────────────────────────┘

┌───────────┬─────────────────────────────────────────────────────────────────┐
│ ID        │ SAI - Semantic Alienation Index                               │
│ Question  │ Do those we serve understand our words?                       │
│ Threshold │ 0.7                                                           │
│ Formula   │ Semi-mathematical with keyword weights                        │
└───────────┴─────────────────────────────────────────────────────────────────┘

                       QUICK START GUIDE
================================================================================

1.  Clone the repository:
    git clone https://github.com/yourusername/GOE_OS.git
    cd GOE_OS

2.  Install dependencies:
    pip install -r requirements.txt

3.  Run the application:
    uvicorn main:app --reload

4.  Access the API documentation:
    http://localhost:8000/docs

5.  Run with Docker:
    docker-compose up -d

6.  Access services:
    - GOE OS: http://localhost:8000
    - Prometheus: http://localhost:9090
    - Grafana: http://localhost:3000 (admin/admin)
